# Breaking Enemies

いわゆるタワーディフェンスゲーム。

HSP 3.31 RC1 製。

2012年の高専祭で展示しました。

以下展示当時の遊び方説明です。

## 基本的な操作方法 (Stage 1開始前にも表示されます)
1. "Play Game"を左クリックします。
   ※"Option"をクリックすると難易度を選択できます。
2. 遊びたいステージを↑キーと↓キーで選び，Enterキーを押します。
   ※ステージ名の前に"◆"がついているものが選択されます。
3. ステージの説明が表示されるので，Enterキーを押します。
   ※ESCキー(キーボード左上)を押すと2.に戻ります。
4. ゲーム説明が表示されます。
   ※スキップできません。
5. カウントダウンが始まりますが，"Defender"(城を守る建物)を設置できます。
6. カウントダウンが終わると敵が出現し始めます。
7. すべての敵を倒せばクリアです。
   逆にすべての敵を倒す前に城の耐久値が0になるとゲームオーバーです。
8. クリアまたはゲームオーバーになると2.に戻ります。
   ※クリアするとスコアが表示されます。
